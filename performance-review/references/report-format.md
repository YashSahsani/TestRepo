# Report Format

Daily response:

1. `Summary`: 3-6 bullets of work completed or advanced.
2. `Evidence`: source-linked bullets with short snippets.
3. `Collaboration`: meetings, reviews, Teams threads, and decisions.
4. `Blockers / Follow-ups`: only if evidence supports them.
5. `Review-ready bullets`: accomplishment phrasing suitable for performance reviews.

Date-range response:

1. `Executive summary`: concise roll-up across the range.
2. `Themes`: grouped work areas with dates.
3. `Goal / OKR alignment`: only when goals or OKRs are provided.
4. `Evidence highlights`: strongest source-linked examples.
5. `Potential review bullets`: impact-oriented phrasing.

Style:

- Keep claims evidence-backed.
- Mention missing sources or stale cache data.
- Prefer dates and links over vague phrasing.
- Do not overstate impact when evidence only shows activity.
