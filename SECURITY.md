# Security

This prototype may process performance-review evidence and employee-related context. Treat all real company data as confidential.

## Do not commit

- API keys, tokens, secrets, or private credentials.
- Real employee performance data.
- Exported company documents unless explicitly approved.
- Generated cache output containing sensitive content.

## Reporting issues

Report security or data-handling concerns to the project owner or the internal security contact before sharing details broadly.
